## About

`Bonsai.Sleap` provides operators for real-time multi-pose tracking using [SLEAP-NN](https://nn.sleap.ai/) with [ONNX Runtime](https://onnxruntime.ai/) inference in the [Bonsai visual reactive programming language](https://bonsai-rx.org).

## How to Use

To use `Bonsai.Sleap` for visual reactive programming, please install this package using the [Bonsai package manager](https://bonsai-rx.org/docs/articles/packages.html).

## Additional Documentation

For additional documentation and examples, refer to the [official Bonsai documentation](https://bonsai-rx.org/sleap).
