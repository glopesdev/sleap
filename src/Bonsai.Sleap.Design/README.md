## About

`Bonsai.Sleap.Design` provides reactive visualizations for markerless multi-pose estimation using [SLEAP-NN](https://nn.sleap.ai/).

## How to Use

To use `Bonsai.Sleap.Design` for visual reactive programming, please install this package using the [Bonsai package manager](https://bonsai-rx.org/docs/articles/packages.html).

## Additional Documentation

For additional documentation and examples, refer to the [official Bonsai documentation](https://bonsai-rx.org/sleap).
